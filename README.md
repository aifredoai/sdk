# Fred SDK

Fred is an AI business development agent you self-host on your own server.

- Finds and researches prospects
- Sends outreach emails via your Gmail
- Books meetings via Google Calendar
- Manages your sales pipeline
- Runs deep research via sub-agents

## Quick Install
```bash
git clone https://github.com/aifredoai/sdk.git fred
cd fred
cp .env.example .env
# Add your Anthropic API key and domain
docker compose up -d --build
```

Full guide: [SELF_HOSTING.md](./SELF_HOSTING.md)
